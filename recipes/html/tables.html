Tables

Overview

Tables should be used to display data in two-dimensional lists. Do not use tables to structure you web page.

Table data width and height without CSS

  • To define the widths of cols without CSS, use the width attribute.
  • If you use n cols (table data elements), you only to define only n-1 element widths. The last width is automatically set.
  • You only have to define the table datas of one row. The rest of the table will align to the widths.
  • Of course you can set the height of tables with the height attribute, but try to avoid that.
