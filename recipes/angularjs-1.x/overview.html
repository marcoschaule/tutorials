Overview

AngularJS Features

The main features of AngularJS 1.x are:

  • Template
  • Expressions
  • Model
  • View
  • Scope
  • Module
  • Controller
  • Service
  • Filter
  • Directive
  • Component
  • Compiler
  • Linker
  • Injector

Filter

A filter is a function to format values of expressions.

Build-in filters are:
currency, date, filter, json, limitTo, lowercase, number, orderBy, uppercase

Directive

A directive is a function that extends HTML with custom elements and attributes, which should be used for DOM manipulations.
